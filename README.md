# DevOps Portfolio Project

A containerized microservices demo with React, Node.js, PostgreSQL, AWS EKS, and Terraform.

## Prerequisites

- Node.js >= 16  
- Docker  
- AWS CLI configured  
- Terraform >= 1.0
